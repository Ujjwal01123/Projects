Rest API -
    Product Schema- title , image url,description, price, countInStock, rating and reviewNo

/api/v1/products:
    ->GET '/api/v1/products'  : get All products
    ->POST '/api/v1/products'  : add new product
    ->GET '/api/v1/products/:id'  : get product with id
    ->PUT '/api/v1/products/:id'  : update product with id
    ->DELETE '/api/v1/products/:id'  : delete product with id


/api/v1/users:
    ->POST '/api/v1/users' : signup
    ->POST '/api/v1/users/login' : login
